# Servidor de desarrollo 
___

1. ### ***Ejecutar el siguiente comando para conectar el servidor***
   
Ingresar a la carpeta donde creamos nuestro proyecto de django.

```
cd premiosplaziapp
```
Ejecutar el siguiente comando.

```
python manage.py runserver
```
